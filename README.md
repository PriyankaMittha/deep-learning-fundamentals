# 🔥 Deep Learning Fundamentals

## 📌 Overview

This repository contains implementations of fundamental deep learning concepts using Python. It focuses on building a strong foundation by explaining core ideas with simple code, visualizations, and examples.

---

## 📂 Projects Included

### 1️⃣ Perceptron Implementation

* Implementation of a single-layer neural network
* Binary classification using manually created dataset
* Decision boundary visualization
* Explanation of XOR limitation

## 🧠 Key Concepts Covered

* Linear separability

## 🎯 Key Takeaways

* Perceptron is the foundation of neural networks
* Works only for linearly separable data
* Cannot solve XOR problem
* Activation functions introduce non-linearity
* ReLU is widely used in deep learning
* Softmax is used for multi-class classification
